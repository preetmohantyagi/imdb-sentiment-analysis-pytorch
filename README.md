# imdb-sentiment-analysis-pytorch
Deep Learning based IMDB movie review sentiment classification using PyTorch RNN with TF-IDF feature extraction and NLP preprocessing.

## Dataset

This project uses the IMDB Movie Reviews Dataset.

You can download it from Kaggle:

https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

After downloading, place the dataset in the project root directory:

```
IMDB Dataset.csv
```
